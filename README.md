# Data Marketplace

The Proof of Concept Data Marketplace built using MAM and IOTA.

## Getting started

### To run for development.

```javascript
yarn install-all

yarn start
```

or

```javascript
npm run install-all

npm run start
```

### To run for Production:

```javascript
yarn install-all

yarn deploy PROJECT_NAME
```

or

```javascript
npm run install-all

npm run deploy PROJECT_NAME
```
