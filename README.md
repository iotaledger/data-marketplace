# Data Marketplace

The Proof of Concept data markertplace built using MAM and IOTA.

## Getting started

To run for development (HMR etc)

```javascript
yarn

yarn dev
```

To run for Production:

```javascript
yarn

yarn build

yarn start
```


### Access logs

* Request access to Firebase console

* Proceed to `Functions` -> `Logs`

* Change `log level` to `Info`
